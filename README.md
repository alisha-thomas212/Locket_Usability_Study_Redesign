# Locket_Usability_Study_Redesign

Project Overview
Conducted end-to-end UX research and redesign for the Locket Widget photo-sharing app. Identified critical usability flaws (inefficient workflows, confusing icons, lack of feedback, hidden features) through user observations, then designed, prototyped, and evaluated two high-fidelity Figma solutions. The winning design achieved a 41% faster task completion and 100% task success rate with statistical significance (p = .007).

My UX Contributions Across the Full Design Process
- Research & Discovery: Think-aloud observations, user interviews, deductive coding, field notes
- Synthesis:	Personas (2), Cognitive Task Analysis (CTA), System Usability Scale (SUS) scoring
- Prototyping: Storyboarding (2 concepts), wireframing, high-fidelity Figma prototypes
- Evaluation:	Comparative usability study (N=6), Cognitive Walkthrough, KLM-GOMS analysis
